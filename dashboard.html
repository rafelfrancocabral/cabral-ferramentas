Formato do Ficheiro

O ficheiro deve ter os seguintes cabeçalhos, nesta ordem:

Código, Descrição, Unidade, Marca, Categoria, Preço, Estoque, Palavras Chaves - (separar com virgula)

Tipos de dados:

  • Código → texto
  • Descrição → texto
  • Unidade → texto
  • Marca → texto
  • Categoria → texto
  • Preço → número decimal (ex: 49,90)
  • Estoque → número inteiro
  • Palavras Chaves → separadas por vírgula

Regras:

  • Máximo 1000 produtos por importação
  • Primeira linha deve conter apenas os cabeçalhos
  • Código não pode estar duplicado
  • Formatos suportados: CSV e XLSX
